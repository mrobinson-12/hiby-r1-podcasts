# MP3 Sync for Hiby R1

This is a simple tool I made to sync my podcasts and music to my mp3 player, the Hiby R1. It is a tool with a python backend and html frontend which can be hosted pretty much anywhere. It is designed to be used with the upload via network on the player but USB support is coming soon. There is lots of bugs but nothing major that I have found.
